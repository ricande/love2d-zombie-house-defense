# Animation Source Pipeline

This folder stores source animation files (for example GIFs) before conversion.

Suggested workflow (later):

1. Put raw source files in:
   - `assets/source/animation/player/`
   - `assets/source/animation/enemies/`
   - `assets/source/animation/ui/`
2. Use ImageMagick to extract/pack frames into sprite sheets.
3. Save exported runtime sprites to:
   - `assets/graphics/player/`
   - `assets/graphics/enemies/`
   - `assets/graphics/ui/`
